# Shared Utilities and Constants

This directory contains shared code that can be used across both backend and frontend.

## Contents

- `constants/` - Shared constants
- `types/` - Shared type definitions
- `utils/` - Utility functions

## Note

For a monorepo structure, consider using:
- **Backend**: Java shared code in a separate module
- **Frontend**: TypeScript/JavaScript shared code
- **API**: Shared OpenAPI schemas and DTOs
